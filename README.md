# 興隆社宅 D2 青年創新回饋計畫 (Xinglong D2 Youth Innovators Living)

這是興隆社宅 D2 區青年創新回饋計畫的官方網站專案。本計畫旨在透過青年力量，為社宅與周邊社區注入創新活力，串聯社區關係，營造共融生活。

## 專案簡介

本網站用於展示計畫內容、活動紀錄、生活故事以及相關募資/募知資訊。
網站採用現代化靜態網頁技術建置，以提供最佳的瀏覽體驗與視覺呈現。

### 技術架構

*   **框架**: React 19 + Vite
*   **語言**: JavaScript (ES6+)
*   **樣式**: CSS Modules + CSS Variables
*   **路由**: React Router v7
*   **部署**: GitHub Pages

## 本地開發 (Development)

1.  Clone 專案：
    ```bash
    git clone https://github.com/elmerlu/XLD2YILiving.git
    ```
2.  安裝依賴：
    ```bash
    npm install
    ```
3.  啟動開發伺服器：
    ```bash
    npm run dev
    ```

## 部署 (Deployment)

本專案配置為部署至 GitHub Pages：
```bash
npm run deploy
```
詳細部署流程請參考 [DEPLOY.md](./DEPLOY.md)。
